This project looks at air quality data using Python. I used a dataset with PM2.5 values and analyzed how pollution changes over time.

What I did
- Loaded the dataset from a CSV file
- Cleaned the data by removing missing values
- Converted dates into a usable format
- Calculated basic stats like average, min, and max PM2.5
- Grouped data by month to see trends
- Created graphs to visualize the results

Tools
- Python
- pandas
- matplotlib

Files
- air_quality_analysis.py, main script
- air_quality.csv, original dataset
- cleaned_air_quality.csv, cleaned data
- PNG files, graphs from the analysis

What I found
- PM2.5 levels stayed relatively low most of the time, but there were occasional spikes where pollution increased sharply. One major spike stands out around 2020, suggesting a short period of unusually high pollution. Monthly averages stayed more stable, usually between about 6 and 12, which shows that extreme spikes were not constant but still influenced overall trends. Overall, the data shows that air quality is generally consistent with periodic pollution events rather than steady increases.
